Basics: tag entities across ten languages

The simplest thing this model does: read text — Arabic, German, English, Spanish, French, Italian, Latvian, Dutch, Portuguese or Chinese — and colour the named things with a confidence for each. Switch languages with the buttons; the model never changes.

backend – latency – entities –

The model was fine-tuned on news-style text across ten high-resource languages, so it's sharpest on public figures, companies, places and dates, and hazier on rare or invented names. Hover any highlight to see how confident it was.
