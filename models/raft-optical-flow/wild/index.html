Big motion & the textureless trap

Large motion: RAFT's all-pairs correlation is built to track big displacements that break simpler (small-motion) methods — crank the shift right up and watch the recovered flow keep matching it. An honest note on the classic aperture problem (flat, textureless regions where motion is ambiguous): a uniform pan like this one moves the whole frame the same way, so RAFT propagates a confident answer even into low-texture areas — the aperture failure really shows up with differential motion (an object moving against a still background), which needs a real video pair, not a synthetic pan.
