Bird species classification — which bird is this?

Drop in a photo of a bird and the model names its species — one of 525 — with the full confidence ranking. This is fine-grained recognition (individual species, not just "bird"), the way a food classifier tells ramen from pho. Everything runs on your device — the photo never leaves the tab.

Why it matters

Fine-grained recognition — telling one species from a look-alike — is where general classifiers fall short and dedicated models shine. Doing it in the browser turns any phone into a field guide that never uploads your photos.

  • Birdwatching — snap a bird and get a species ID (and the runners-up) instantly, offline.
  • Citizen science — pre-label sightings before submitting to a survey.
  • Education — a learning tool that shows how confident the model is, and where it's unsure.
  • Accessibility — name what's in a photo for someone who can't see it clearly.

It knows 525 species and is strongest on a clear, centred bird. Look-alikes and unusual angles can fool it — the confidence bars make that visible (a spread-out ranking means "not sure"), and it will confidently pick the nearest of its 525 classes even for a bird it doesn't actually know.

How the API works

It's one image-classification call:

import { pipeline } from "@huggingface/transformers";

const classify = await pipeline("image-classification",
  "dennisjooo/Birds-Classifier-EfficientNetB2");
const out = await classify(birdImage, { top_k: 5 });
// → [{ label: "AMERICAN FLAMINGO", score: 0.99 }, … ]

The EfficientNet-B2 resizes and normalises the image, then scores the 525 species. It runs in a Web Worker so the page stays responsive.
