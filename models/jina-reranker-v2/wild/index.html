Wild: rank code against a plain-English request

The skill that sets v2 apart from most rerankers: it was trained on code and function-calling relevance, not just prose. Describe what you want in plain English and give it candidate code snippets or function signatures — it scores which one actually implements your request. This is exactly the ranking step behind code search and tool/function selection in an agent. Pick a task or write your own.

Notice the ranking uses meaning, not shared words: the correct snippet wins even when a distractor shares more identifiers with your request. That is why a cross-encoder trained on code beats keyword search for code retrieval and function selection — and it is running entirely on your device.
