Practical: sky replacement

Semantic classes are directly useful: if you know which pixels are "sky," you can replace them. SegFormer labels the whole scene, then this page cuts out one class — sky by default — and drops a new backdrop behind everything else. Pick any class to replace, and any sky to drop in. All on-device.
