Multi-model: multilingual alt-text pipeline

Two models, one job. Moondream2 (vision-language, WebGPU) writes an English description of the image, then OPUS-MT (a small translation model, on CPU/WASM) renders it in the language you pick — an end-to-end, on-device pipeline that produces alt-text for a multilingual audience with no server.
