Wild: one query, a corpus in many languages

Because EmbeddingGemma is trained over 100+ languages, it maps meaning into a single shared vector space no matter the language. So an English query can rank a French, Spanish, German, Japanese or Arabic passage by what it means — no translation step, no per-language index. Type a query in any language and search the mixed corpus below.

Notice the two coffee-with-milk passages (Spanish and Arabic) cluster together and rise for a coffee query, while the eclipse passages (English and French) win for the eclipse query — matched by meaning across scripts, not by shared words. That is a multilingual embedding space doing its job.
