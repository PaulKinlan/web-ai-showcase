Basics: how close are two passages?

The simplest thing an embedder does: turn each passage into a 768-number vector, then measure the angle between them. Cosine 1.00 means "same direction / same meaning", near 0 means "unrelated". EmbeddingGemma is instruction-prompted, so the toggle below lets you feel the difference between embedding a line as a document vs as a search query.

Off = both embedded as documents. EmbeddingGemma's prefixes nudge query vs document vectors.

Cosine is scale-free: it only cares about direction, so passage length barely matters — a one-line question and a paragraph-long answer can still score high if they mean the same thing. That is exactly what makes embeddings good at matching short queries to long documents.
