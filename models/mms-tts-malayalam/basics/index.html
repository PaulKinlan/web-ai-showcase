Basics: type a Malayalam line, hear it

The simplest thing an end-to-end TTS model does: text in, waveform out. Type some Malayalam (or pick an example), press സംസാരിക്കുക, and the audio plus its real 16 kHz waveform appear below — all synthesised on your device, in native Malayalam script.
