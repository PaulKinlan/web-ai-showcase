Wild: become someone else

The system prompt is a personality dial. Same weights, same GPU — but tell the model who to be and the whole conversation changes character. Pick a persona (or write your own), then chat. It's a vivid way to feel how much of an assistant's "voice" is just instructions.

Pick a persona, load the model, and start talking.
