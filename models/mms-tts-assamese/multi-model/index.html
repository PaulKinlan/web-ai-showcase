Multi-model: translate, then speak Assamese

Two models, one pipeline, zero servers. NLLB-200 (No Language Left Behind, English→Assamese) translates what you type into native Assamese script, then MMS-TTS Assamese reads the translation aloud. We pair with NLLB here — not M2M100 — because NLLB-200 covers Assamese (asm_Beng), which M2M100's 100-language set does not. Type English, get spoken Assamese — entirely on your device.
