Wild: odd-one-out & gallery clustering

Two things that only feel possible once a label-free vision model runs in a tab. Drop in a set and let DINOv2 spot the odd one out — the image least like the rest. Then cluster a gallery by look alone: a moodboard, a shoot, a folder of screenshots, grouped with no labels.

Each image's average similarity to the others is computed; the lowest is the outlier.
