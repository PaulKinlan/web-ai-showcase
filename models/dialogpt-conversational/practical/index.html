Practical: quick-reply suggestions

The "smart reply" chips in your messaging app, but fully on-device. Paste a message you received and DialoGPT samples three candidate replies — because sampling is random, each run gives different options. Pick one, tweak it, done. No message ever leaves the tab.

Each suggestion is a real DialoGPT sample from the same message, so quality varies — that's the point of showing three. In a real product you'd filter and rank them; here they're raw so you can see the model's range.
