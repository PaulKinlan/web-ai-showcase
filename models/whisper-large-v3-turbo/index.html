Whisper large-v3-turbo — high-accuracy transcription, on your device

Turbo is the full large-v3 encoder with a pruned 4-layer decoder (large-v3 has 32) — near large-v3 accuracy at several times the decode speed. It handles 99 languages, auto-detects the one being spoken (with a real probability), can force a language, and can translate any speech straight to English — all in your browser, nothing uploaded.

This is a big model. On WebGPU it is fast and compact; on CPU/WASM it still runs and produces the same text, just slower (an 11-second clip can take ~30 s to decode headless). The page reports the real backend, dtype, and timing it used.

Why it matters

Base Whisper is great until the audio gets hard — heavy accents, background noise, or a less-common language. Turbo brings large-v3-grade robustness to the browser without the full large-v3 decode cost, so high-quality, multilingual transcription becomes a private, offline, zero-cost capability of your page.

  • Multilingual transcription — auto-detect and transcribe 99 languages with near-flagship accuracy, on-device.
  • Speech translation — turn speech in any supported language directly into English text, no separate translation model.
  • Hard-audio robustness — accents, noise, and domain jargon that trip up smaller models.
  • Private, offline, free — meeting notes, interviews, field recordings that never leave the device and cost nothing per minute.

How the API works

Transcription is one pipeline call; language and task are options:

import { pipeline } from "@huggingface/transformers";

const asr = await pipeline(
  "automatic-speech-recognition",
  "onnx-community/whisper-large-v3-turbo",
  { device: "webgpu", dtype: "q4f16" }, // WASM + q8 fallback when no GPU
);

// audio = a 16 kHz mono Float32Array (decode with Web Audio, then resample)
const out = await asr(audio, {
  language: "es",        // force a language, or omit to auto-detect
  task: "translate",     // "transcribe" (same language) | "translate" (→ English)
  return_timestamps: true,
  chunk_length_s: 30,
});
// out.text   → the transcript (or English translation)
// out.chunks → [{ text, timestamp: [start, end] }, …]

For the "detected language" readout, this page runs one extra decoder step and takes a softmax over Whisper's language tokens — the model's real language distribution. The main-thread work is only audio decode + drawing; the detection and transcription both run in a Web Worker.
