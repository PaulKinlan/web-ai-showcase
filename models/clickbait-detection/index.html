Clickbait detection — is this headline baiting you?

"You won't believe what happened next" tells you nothing except that someone wants the click. This DistilBERT headline classifier decides whether a headline is clickbait / manipulative — curiosity gaps, withheld payoffs, emotional bait — or a straight, informative headline that states the news. It's a signal for media-literacy triage, not a truth verdict. Type a headline below and watch the bait meter move, all on-device.

See inside — two competing probabilities

This is the model's raw softmax: two numbers, manipulative and legitimate, that sum to 1. Clickbait has a fuzzy surface signal — the model leans on curiosity-gap patterns it saw in training (withheld payoffs, "you won't believe", list numbers, emotional superlatives), so it's strong on stock bait and genuinely uncertain on punchy-but-real headlines. The chips below the meter highlight cue phrases the UI spotted — a teaching aid, not the model's actual features. When the two bars sit near 50/50, that uncertainty is the finding: treat the score as a triage cue, never a truth label.

Why it matters

Clickbait is the front door of low-quality and manipulative media. A cheap, private, on-device flag lets you screen headlines as they're rendered — in a reader, a feed, a browser extension — without shipping anyone's reading habits to a server.

  • Feed hygiene — down-rank or badge curiosity-gap headlines in a reader so the straight news rises.
  • Media-literacy triage — surface a "bait" cue next to a headline as a prompt to read critically, not as a censor.
  • Newsroom self-check — warn an editor that a draft headline reads as bait before it ships.
  • Dataset cleaning — pre-filter a headline corpus into bait vs straight for downstream analysis.

Use cases

Basics

Type a headline, get a clickbait vs straight verdict live, and compare bait/neutral rewrites of the same story side by side.

Practical

Run a feed of headlines through a bait pre-check and triage them — badge the clickbait as a media-literacy signal, not a verdict.

Wild

Probe the borderline — punchy-but-real headlines, deadpan bait, curiosity gaps that turn out to be honest — and see where the model wavers.

Multi-model

Chain a clickbait flag with sentiment: separate loud emotional bait from calm, negative-but-straight reporting.

How the API works

Binary classification is one call; ask for both scores with top_k:

import { pipeline } from "@huggingface/transformers";

const classify = await pipeline(
  "text-classification",
  "CoolpantsMcBadass/headline-classifier",
  { dtype: "q8" },
); // downloads + caches ~67 MB once

const out = await classify("You won't believe what happened next", { top_k: 2 });
// → [{ label: "manipulative", score: 0.99 }, { label: "legitimate", score: 0.01 }]

A default problem_type means Transformers.js applies a softmax over the two classes, so manipulative and legitimate sum to 1. It runs in a Web Worker so typing never janks.
