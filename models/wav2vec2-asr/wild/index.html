Wild: scrub the frames

A seq2seq model only hands you finished text. A CTC model exposes its raw, frame-by-frame guess — so let's scrub through it. Transcribe a clip, then drag the slider (or use arrow keys) to move through the ~20 ms frames and read exactly which character the model thought was there at each instant, blanks and all.
