Practical: semantic search & near-duplicate clustering

Two things you'd actually ship. First, meaning-based search that finds the right passage even when it shares no keywords with the query — with a naive keyword score alongside so you can see Nomic win. Second, label-free clustering that collapses "same thing, different words." Both run entirely in the tab, no vector database.
