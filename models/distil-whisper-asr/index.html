Distil-Whisper — the same Whisper, distilled to run faster

Distil-Whisper is Whisper with its decoder shrunk by distillation: it keeps the full Whisper encoder but runs far fewer decoder layers, so it writes the transcript in less wall-clock time at close accuracy (English-only). Same seq2seq model, same timestamps — just fewer passes per token. Record, drop a file, or pick a sample and watch the real-time factor.

Why it matters

Whisper is accurate but its decoder runs a full stack of layers for every token, which costs time on-device. Distil-Whisper trades a little breadth (English-only, distilled) for markedly faster wall-clock transcription — the same private, offline, no-upload capability, but snappier for captions and batch jobs.

  • Faster captions — subtitle recorded audio/video in less time, entirely client-side.
  • Batch transcription — churn through a folder of voice notes or clips quickly, no per-file API bill.
  • Cheaper interactivity — a lighter decoder means lower latency between "stop talking" and "see text".
  • Edge & offline — Whisper-grade English transcription where bandwidth or privacy rules out the cloud.

How the API works

It's the exact same pipeline call as Whisper — only the model id changes:

import { pipeline } from "@huggingface/transformers";

const transcribe = await pipeline(
  "automatic-speech-recognition",
  "distil-whisper/distil-small.en", // distilled: fewer decoder layers, English-only
  { device: "wasm", dtype: "q8" },  // WebGPU when a real adapter exists
);

// audio = a 16 kHz mono Float32Array (decode with Web Audio, then resample)
const out = await transcribe(audio, {
  return_timestamps: true, // segment-level timings — seq2seq keeps them
  chunk_length_s: 20,
  stride_length_s: 3,
});
// out.text   → the transcript
// out.chunks → [{ text, timestamp: [start, end] }, …]

This page decodes the mic/file/sample to 16 kHz mono with the Web Audio API on the main thread, then transfers just the Float32 samples to a Web Worker where the pipeline runs — so the model never blocks the interface. It times the call and divides by the clip length for a real real-time factor. Because it's still Whisper under the hood, the API surface is identical — you swap the model id and get a faster, English-only transcriber.
