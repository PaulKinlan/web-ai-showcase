Wild: visual grounding — point at what it sees

Something you couldn't do before this ran in a browser: ask the model where something is, and it answers with coordinates. Qwen2.5-VL is trained to emit bounding boxes, so a plain-language request — "find every piece of fruit" — comes back as boxes we draw straight onto the image. No separate object detector, no server: language-guided localisation, entirely on your GPU.

How the coordinates line up: the model emits pixel coordinates in the resized image it actually saw (a multiple of 28 on each side, native aspect ratio). We report that processed size and scale the boxes back onto the displayed picture. If the model returns text with no parseable boxes, we show the raw output and draw nothing — never a fabricated box.
