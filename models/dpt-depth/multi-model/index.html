Multi-model: how far away is each thing?

DPT answers "how far is every pixel?" SegFormer answers "what is every pixel?" — a semantic class (sky, floor, person, plant…). Line the two maps up and average depth within each class, and you get scene understanding neither model has alone: a ranked list of the things in the photo, nearest to furthest. Two tiny models, one device, no server.
