How similar are two snippets?

The fundamental capability, felt directly: the model turns each snippet into a 768-d unit vector, and cosine similarity — the dot product — is one number from −1 to 1 that says how close in meaning they are. Edit either side and watch it move. Try the "same algorithm, different language" example: a keyword diff would call them unrelated; the embedder does not.
