Retrieve the right code, then explain it

This is retrieval-augmented understanding, entirely on-device. Stage 1: Jina's code embedder ranks a snippet bank against your English question and pulls the best match. Stage 2: that snippet is handed to TinyLlama-1.1B — a small chat LLM — which explains, in plain language, what the code does. The embedder finds; the LLM explains. Both run in your browser, so a private codebase stays private.
