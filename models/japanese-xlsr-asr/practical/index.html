When was each character said?

A seq2seq model gives you text but not timing. CTC pins every character to a ~20 ms frame, so the frame each character starts and ends at is a real byproduct of decoding — no extra model. Japanese has no spaces between words, which makes the character the natural alignment unit. Below, each transcribed character carries its start–end time. Click one to hear just that moment.

What's happening

The worker records the frame index at which each Japanese character was emitted. Multiply by the frame duration (audio length ÷ frame count) and you get wall-clock timings for every character. This is genuine forced alignment straight from the acoustic model — no separate aligner, and it works even though Japanese is written without word spaces.
