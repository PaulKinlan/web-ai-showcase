Wild: dense patch correspondence between two images

Beyond one-number-per-image similarity: the model gives a feature vector for every 14×14 patch, and because I-JEPA was trained to predict masked-region representations, those patch features are semantically organised enough to match regions across two images. For each patch of image B we find its nearest patch in image A and paint it that patch's position colour — so the same part (an ear, a wheel, a horizon) glows the same colour in both. This is the raw material of registration, tracking, and part alignment, running entirely on-device.
