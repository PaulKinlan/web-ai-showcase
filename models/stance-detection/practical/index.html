Practical: map a debate

A motion and a pile of arguments — comments, review snippets, retrieved passages. Score each against the claim on-device and drop it into a for / against / unclear board. This is the workhorse of stance: turning an unsorted heap of opinions into a structured map a human can act on, privately, in one pass.

A map, not a judge

The board is a first pass, not a ruling. Off-topic lines (the farmers' market) land in "unclear", and a subtly-hedged argument can straddle the line — which is exactly where a human should look. Because it runs locally, none of the arguments leave the device.
