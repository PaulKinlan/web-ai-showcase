Wild: the cross-language parity probe

A single multilingual model promises the same judgment in every language — but it never quite delivers. Take one sentence, translate it into eight languages, and score all eight. The meaning is identical, so the toxic probability should be too. Where it drifts — and where a clearly hostile line slips under the threshold — is exactly where the model is weaker. The reported test F1 next to each language tells you where to expect trouble (Arabic and German are the shakiest).

A "miss" is a line whose verdict (at threshold 0.50) disagrees with the meaning — a hostile sentence scored below 0.50, or a friendly one above it. This is the honest reason a responsible multilingual moderation pipeline routes low-confidence and weak-language cases to human review rather than trusting one number.
