Multi-model · captioner → CLIP

Caption an image, then ground it across languages

Two models in a row. First ViT-GPT2 writes an English caption. Then jina-clip-v2 grounds it — scoring the caption against the image — and you can add the same idea in other languages to confirm the grounding holds cross-lingually. A generator and a multilingual matcher, both on your device.

Heads up: this page loads two models — the captioner (~250 MB) and jina-clip-v2 (~874 MB). Best on desktop with a good connection; cached after the first load.

How the two models connect

const caption = (await captioner(image))[0].generated_text;      // ViT-GPT2 → English text
const labels  = [caption, ...paraphrasesInOtherLanguages];

const img = await clip({ pixel_values });                        // jina-clip-v2
const txt = await clip(tokenizer(labels));
// cosine(img, txt[i]) → how well each phrasing grounds to the picture
