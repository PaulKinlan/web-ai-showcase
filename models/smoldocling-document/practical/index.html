DocTags are structured, so they convert cleanly. Here SmolDocling reads a page and we turn its DocTags into Markdown — headings become #, tables become Markdown tables, formulas become $$…$$ — that you can copy straight out. Nothing leaves the device; this is what a private, offline document converter looks like.

What's happening

SmolDocling emits DocTags; we parse each element (heading, paragraph, <otsl> table, <formula>) and render it as the matching Markdown. Because the model preserved the document's structure, the conversion is a straight mapping — not a fragile text-reflow guess. Swap in your own scan and you have an offline OCR-to-Markdown tool.
