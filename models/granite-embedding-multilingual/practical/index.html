Practical: cross-lingual search & near-duplicate detection

Two workhorses. Semantic search: rank a set of passages by meaning against a query, not by keyword overlap — and because the model is multilingual, the query and the passages can be in different languages. Dedup: find pairs that say the same thing in different words — or different languages — so you can collapse them. Both embed each passage once and reuse the vectors.

Near-duplicate pairs

Pairs whose cosine is above the threshold — candidates to merge.
