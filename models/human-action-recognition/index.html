Human action recognition

The built image classifiers name the object in a photo. This one names the activity: given a picture of a person, it predicts what they're doing — one of 15 everyday actions (running, drinking, cycling, dancing, using a laptop, …). It's a ViT-Base fine-tuned for human action recognition. Below it reads real licensed photos; drop in your own to try it. Runs entirely on your device.

At a glance

Object vs action

A plain image classifier trained on ImageNet would answer "person" or "bicycle". This model answers the harder question — what the person is doing — because it was fine-tuned on labelled actions rather than objects. It reads a single still, so ambiguous poses get lower-confidence guesses; the bars show exactly how sure it is. Everything runs locally.
