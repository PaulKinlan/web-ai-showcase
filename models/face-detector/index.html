FaceDetector — box every face

MediaPipe's FaceDetector (BlazeFace short-range) finds faces in an image or video and returns, for each one, a bounding box, 6 keypoints (eyes, nose tip, mouth, ears) and a confidence score. It's a tiny .tflite model — about 230 KB — that runs on-device via WebAssembly (with a GPU delegate when available). Drop a photo or turn on your webcam.

Why it matters

A face box is the first step in almost every camera feature. FaceDetector is small and fast enough to run every frame on a phone, so "is there a face, and where?" becomes a free local primitive — no cloud vision API, no per-call cost, nothing leaving the device.

  • Privacy tooling — blur or redact faces before an image is shared or uploaded, client-side.
  • Smart capture — auto-crop to faces, frame a group shot, or trigger the shutter when everyone's in frame.
  • Analytics without identity — count faces for footfall or engagement without ever recognising anyone.
  • Detector stage for a pipeline — gate a heavier model (landmarks, recognition) so it only runs where a face is.

How the API works

MediaPipe Tasks loads the model asset (a .tflite bundle for this detector) and gives you a detector you call per image or per video frame:

import { FilesetResolver, FaceDetector } from "@mediapipe/tasks-vision";

const vision = await FilesetResolver.forVisionTasks(".../wasm");
const detector = await FaceDetector.createFromOptions(vision, {
  baseOptions: {
    modelAssetPath: ".../blaze_face_short_range.tflite", // ~230 KB
    delegate: "GPU",                                     // or "CPU"
  },
  runningMode: "IMAGE",                                  // or "VIDEO"
});

const result = detector.detect(imageElement);
// result.detections → [ {
//   boundingBox: { originX, originY, width, height },   // pixels
//   keypoints:   [ {x, y} × 6 ],                        // normalized 0–1
//   categories:  [ { score } ],
// }, … ]

This page loads the model through the showcase's shared auto-init loader and draws the boxes on a canvas at the image's natural resolution; for the webcam it switches the task to VIDEO mode and calls detectForVideo each frame.
