Why chain three stages

Each component is weak alone but strong together. T5-SQL knows how to produce SQL from a schema but can't run it or talk about results; SQLite runs SQL but knows nothing about the question; FLAN-T5 can't query a database but is good at reading a small set of facts and answering from them. Composed, they form the same retrieve-then-answer shape as a production text-to-SQL assistant — with nothing leaving the device. The tiny FLAN model answers single-value questions cleanly; multi-row aggregates are where its 80M-parameter ceiling shows, and the page shows you the real rows either way.
