Colourise a grayscale photo

The plain capability: strip a photo to grayscale — exactly what the model sees — then let DDColor invent a full palette. Drag the divider to wipe between before and after.

Run it

◀ grayscalecolourised ▶
Drop / choose your own photo
backend –latency –chroma std –

What you're seeing

The model only outputs colour (chroma) — every edge, texture and highlight in the result is your original photo's luminance, untouched. So it can never blur or distort the picture; it can only get the colour right or wrong. Try a landscape (sky/foliage are easy wins) versus an object it has never seen a canonical colour for (a painted wall, clothing) to feel where it is confident and where it guesses.
