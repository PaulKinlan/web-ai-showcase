Revive an old photo: sharpen, then colour

Old scans are often soft and grey. This pipeline fixes both: NAFNet restores the detail, then DDColor invents the colour. Two models, chained, running entirely on your device — a private one-click restoration for a family archive.

Run it

Two models load below (deblur ~92 MB, colour ~130 MB). Pick a photo — we soften it to mimic an old scan — then run the pipeline: sharpen → colourise.

1 · NAFNet (deblur / restore)
2 · DDColor (colourise)

◀ soft grayscale scansharpened + colourised ▶
deblur –colour –backend –

Both models run on-device; the scan and every restored pixel never leave the tab.

Order matters

Deblur before colourise: DDColor reads luminance to decide colour, so giving it a sharper, cleaner luminance means better-placed colour. Restoration first, colour second — the same order a human retoucher would use.
