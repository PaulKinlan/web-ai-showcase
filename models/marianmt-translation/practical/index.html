Practical: localise UI strings, fast

When you know your target markets, per-pair models are the efficient choice. Paste your interface strings (one per line), pick a target language, and get first-pass translations — the whole batch, from one small model, on-device. A strong draft to hand a reviewer, with no server and no per-string fee.

strings –pair –backend –total time –

Machine translation is a strong first draft, not a substitute for a native speaker — but getting the whole string table translated in seconds, offline, is a real head start. Switch the target to translate the same table into another market's language.
