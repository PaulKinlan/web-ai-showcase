Basics: translate a phrase

The simplest thing the model does: take an English sentence and render it in another language. Type a phrase, choose a pair, translate. Then switch the pair — the first time you pick German, French, or Spanish, that small (~105 MB) model downloads; after that it's instant and offline.

Each pair is its own model — there's no language code to set, the direction is the model. That's the Marian trade: swap M2M100's one-big-model for several small, fast ones, and only ship the directions you need.
