Zero-shot image tagging

Turn the shared space around: instead of ranking images for a phrase, rank phrases for an image. Pick or drop a photo, type any label phrases, and the model scores how well each matches — a CLIP-style zero-shot classifier with labels you invent at runtime, no training, on-device.
