DepthPro — sharp, metric-scale depth from one photo

DepthPro is Apple's high-resolution monocular depth model. Give it one photo and it estimates the distance to every pixel — but unlike a smooth relative map, DepthPro is trained for metric-scale depth with razor-sharp boundaries: crisp object edges and a consistently-scaled depth field. It's a large model that runs on WebGPU; the photo never leaves the tab.

DepthPro outputs a tensor of real numbers, one per pixel — the raw metric-scale depth field. DepthPro vs relative depth (Depth Anything)

Depth Anything is small, fast, WASM-friendly, and gives a relative depth map — great for parallax and blur, but the range is arbitrary and re-normalized per image, and fine boundaries soften. DepthPro trades size and a WebGPU requirement for metric scale (consistent absolute depth) and sharp boundaries — the right tool when you need real distances or a clean cut at object edges. Same one-photo input; different guarantees.

Why it matters

Metric, boundary-accurate depth from a single frame — no stereo rig, no LiDAR — opens up effects and measurements that a relative map can't support:

  • Real distances — a consistently-scaled depth field lets you compare "how far" across the scene, not just "nearer/further".
  • Clean cut-outs & portrait blur — sharp boundaries mean the subject/background split follows real edges, so a depth-driven blur or matte looks crisp.
  • AR occlusion & compositing — place virtual objects at the right depth with edges that don't bleed.
  • 3D from a photo — a high-resolution depth field drives believable parallax and relighting.

One photo in, a sharp colourised depth map out — read distance as colour.

How the API works

Depth is a one-liner with Transformers.js. The pipeline returns both a ready-to-display image and the raw tensor:

import { pipeline } from "@huggingface/transformers";

const estimator = await pipeline(
  "depth-estimation",
  "onnx-community/DepthPro-ONNX",
  { device: "webgpu", dtype: "q4f16" },   // ~572 MB, WebGPU; cached after first load
);

const { depth, predicted_depth } = await estimator(imageURL);
// depth          → RawImage: grayscale, min–max normalized to 0–255 (great for display)
// predicted_depth → Tensor:  the raw per-pixel metric-scale depth (dims = [H, W])

This page colourises depth on a canvas and reads predicted_depth for the real range + distribution, and measures boundary sharpness from the depth map, in "See inside". Inference runs in a Web Worker so estimating a large photo never freezes the page. DepthPro is a WebGPU-class model; the loader shows an honest needs-WebGPU state on devices without a GPU adapter rather than downloading ~572 MB or faking a result.
