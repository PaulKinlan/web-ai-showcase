Multi-model: read then summarise a document

Two models, one job. Qwen2-VL (vision-language, WebGPU) transcribes the text in a document image, then DistilBART (a small summariser, on CPU/WASM) condenses that extract into a one-line digest — an end-to-end, on-device "photo → gist" pipeline with no server.
