Multi-model: fill the mask, then tag the grammar

A two-stage, all-French on-device pipeline. First CamemBERT fills your <mask>. Then a second model — also a CamemBERT, fine-tuned for part-of-speech tagging — reads the completed sentence and labels every word (noun, verb, adjective, preposition…). Same architecture, two jobs, chained locally with nothing leaving the tab.

Models

This demo loads TWO ~110 MB CamemBERT models: the fill-mask model (the one the other demos use, cached if you've run them) and the POS tagger.

1 · Fill-mask model

2 · Part-of-speech model

fill –pos –backend –

The tagger runs on the filled-in sentence, so the chain is honest end-to-end: the first model's output word is literally part of the second model's input.
