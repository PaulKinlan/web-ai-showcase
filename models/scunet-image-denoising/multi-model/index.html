Denoise, then upscale

Upscaling a noisy photo just makes bigger noise. Here SCUNet cleans the image first, then Real-ESRGAN upscales the clean result 4× — so the super-resolver sharpens real signal, not grain. Two restoration models, chained, entirely local.

Both models run on-device; the image and every restored pixel never leave the tab.

Why chain them

Order matters in restoration. Super-resolution amplifies whatever it's given — including noise — so denoising first lets the upscaler spend its capacity on real detail. This is the classic "clean then enlarge" pipeline, running entirely in your browser with no server and no upload.
