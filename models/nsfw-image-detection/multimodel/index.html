Multi-model: a safety gate in front of analysis

Moderation is the gate, not the goal. Here two models run in sequence: the NSFW detector screens each image, and only the ones it rates safe pass through to a second model — ViT image classification — which labels what's in them. Anything withheld never reaches the classifier. That's how you protect a downstream pipeline (labelling, captioning, indexing) from unsafe input. Both models run locally; nothing is uploaded.
