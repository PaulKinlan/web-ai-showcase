Basics: do these two lines mean the same thing?

The plain capability. Embed two sentences into 256-number vectors and take their cosine similarity — 1.0 means "same direction / same meaning", 0 means unrelated. Edit either box and re-run; the answer comes back in microseconds because there is no network to run, only a lookup and an average.
