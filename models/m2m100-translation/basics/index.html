Basics: translate a phrase

The simplest thing the model does: take a sentence in one language and render it in another. Type a phrase, choose the target language, and translate. Then change just the target and translate again — same model, any of 100 languages.

Translation
codes –tokens – backend –time –

M2M100 translates directly between the pair you pick — it doesn't secretly pass through English. Try the same sentence into three different targets and you're using one model, not three.
