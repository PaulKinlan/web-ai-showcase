M2M100 — translate between 100 languages, directly

Most translation models go through English. M2M100 is many-to-many: it translates straight from any of 100 languages to any other, set with a source and target lang code. Pick two languages, type, and watch the translation stream in — all on-device.

Why it matters

On-device translation means you can localise, understand, or bridge languages without sending a single word to a server — private, offline-capable, and free of per-call fees. One 470 MB model covers 100 languages in every direction.

  • Private translation — translate messages, docs, or medical/legal text that can't leave the device.
  • Offline travel — a phrasebook that works with no signal, in any of 100 languages.
  • UI localisation drafts — get first-pass translations of interface strings, then have a human review.
  • Pipeline stage — pair with speech-to-text or text-to-speech for a full spoken translator.

How the API works

Translation is one pipeline call; the language pair is set per call:

import { pipeline } from "@huggingface/transformers";

const translator = await pipeline(
  "translation",
  "Xenova/m2m100_418M",
); // downloads + caches ~470 MB once

const out = await translator("Hello, how are you?", {
  src_lang: "en",   // source language code
  tgt_lang: "fr",   // target language code
});
// → [{ translation_text: "Bonjour, comment allez-vous ?" }]

Change tgt_lang to any of the 100 codes and the same input translates to that language — no separate model per pair. This page streams tokens via a TextStreamer inside a Web Worker so the UI stays responsive.
