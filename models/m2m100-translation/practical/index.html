Practical: localise UI strings, then round-trip check them

Paste your interface strings (one per line), pick a target language, and get first-pass translations. Then the round-trip check translates each result back to English so you can eyeball which ones drifted and need a human — a quick confidence signal, all on-device.

strings –target –backend –total time –

A round-trip that comes back close to the original is a good sign the meaning survived; one that comes back different is a flag to check — it doesn't prove the translation is wrong, but it's where a reviewer should look first. Machine translation is a strong first draft, not a substitute for a native speaker.
