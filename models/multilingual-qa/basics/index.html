Basics: point at the answer, in any language

The simplest thing multilingual extractive QA does: read a passage, take a question — in the same language — and highlight the exact words that answer it. Switch languages with a click and watch the same model work on each.

One model, four languages, no translation step. Because the answer is always a span of the passage, the model can only point at what's there — the same grounding guarantee as English extractive QA, extended across languages.
