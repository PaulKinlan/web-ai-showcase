Practical: pull a table off the page

A results table in a PDF is a picture of a grid — you can't sort it, sum it, or paste it into a spreadsheet. Nougat reconstructs the actual |-delimited Markdown table, and this page turns that into a rendered grid plus copyable CSV, all on your device.

Drop a page with a table, or click to choose

The extracted table will appear here.
backend – rows – columns – total –

Because Nougat generates the grid rather than reading cells left-to-right, it keeps rows and columns aligned even when the source has multi-word headers or blank cells — the structure survives the transcription. If no table is found, you'll see the raw Markdown Nougat produced instead.
