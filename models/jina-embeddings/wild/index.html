Wild · feature-extraction · WASM

Sort a pile of passages into themes

Give the model a jumble of unlabeled passages and it will group them — no categories supplied. Embed each one, run k-means over the vectors, and "same topic, different words" collapses into clusters. The 2D map colours each passage by its cluster so you can see the themes separate. Unsupervised topic discovery, entirely on-device.
