Rescue a tiny old photo: enlarge, then colour

A tiny black-and-white thumbnail is small and grey. This pipeline fixes both: Real-ESRGAN enlarges it 4× and reconstructs detail, then DDColor invents plausible colour. Two models, chained, running entirely on your device — a private one-click revival for a small archival scan.

Run it

Two models load below (enlarge ~67 MB, colour ~130 MB). Pick a photo — we shrink it to a tiny grey thumbnail to mimic a small old scan — then run the pipeline: enlarge → colourise.

1 · Real-ESRGAN (enlarge 4×)
2 · DDColor (colourise)

◀ tiny grey thumbnailenlarged + colourised ▶
enlarge –colour –size –backend –

Both models run on-device; the scan and every restored pixel never leave the tab.

Order matters

Enlarge before colourise: DDColor reads luminance to decide colour, so giving it a larger, detail-reconstructed image means better-placed colour than colourising a tiny thumbnail directly. Resolution first, colour second — each model does the job it's best at.
