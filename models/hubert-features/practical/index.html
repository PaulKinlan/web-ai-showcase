Practical: find a sound by example

Query-by-example audio search, on-device. Pick a recording, then drag across the waveform to mark a query region (a word, a phrase, a noise). HuBERT embeds your query and slides a matching window along the whole clip, scoring each position by cosine similarity. The curve shows the match over time; the best segment is highlighted. The query's own location should peak — and so should anywhere the sound repeats.
