The synthetic data factory

Describe the records you want in plain English — "customers with a name, country, plan tier and signup date" — and Granite invents a batch of realistic, structurally-consistent fake rows as a JSON array. It's a genuinely useful enterprise trick (seed a demo, fill a test database, mock an API) turned into a playground: the page parses the JSON and renders it as a table, live, entirely on your device. No real customer data anywhere.

What to notice

The system prompt asks for a JSON array only, with consistent keys across every object; a modest temperature keeps the rows varied but well-formed. Because it runs on-device, you can generate seed data for a prototype with zero privacy risk and no API cost. Ask for a constraint ("emails at example.com", "dates in 2025") and watch it hold across the batch.
