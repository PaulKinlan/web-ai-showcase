Why this pattern works

A system prompt fixes the exact schema for each document type and demands "JSON only, no prose"; greedy decoding keeps the output deterministic — important for a governed pipeline. The page strips any stray code fences and runs JSON.parse() — the honest test of whether the model produced machine-readable output. Granite's enterprise tuning shows up here as reliable adherence to the requested keys. In production you'd validate against a real JSON Schema and re-prompt on a miss; the shape is the same.
