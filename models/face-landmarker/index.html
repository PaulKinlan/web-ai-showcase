FaceLandmarker — 478 points, 52 blendshapes

MediaPipe's FaceLandmarker finds a face in an image or video and returns a dense 478-point mesh, plus 52 blendshape coefficients — ARKit-style expression channels like jawOpen, mouthSmileLeft, and eyeBlinkRight — and a 4×4 head-pose matrix. It's a real downloadable .task model that runs on-device via WebAssembly (with a GPU delegate when available). Drop a photo or turn on your webcam and watch the expression bars move.

Why it matters

Face-mesh tracking used to mean ARKit, a native SDK, or a cloud vision API. FaceLandmarker turns "where is the face, and what expression is it making?" into a local function call in any browser — private, offline, and fast enough for a live camera on a phone. The blendshapes are the real prize: 52 ready-made expression channels you can drive anything with.

  • Avatars & VTubing — drive a 3D or 2D character's face straight from the blendshapes, no rig calibration.
  • Attention & drowsiness cues — read eye-blink and yawn signals for driver-monitoring or focus tools, on-device.
  • Virtual try-on & beauty AR — anchor glasses, makeup, or filters to the exact mesh points.
  • Accessibility — map facial expressions (a raised brow, a smile) to hands-free UI actions.

How the API works

MediaPipe Tasks loads a .task bundle and gives you a landmarker you call per image or per video frame:

import { FilesetResolver, FaceLandmarker } from "@mediapipe/tasks-vision";

const vision = await FilesetResolver.forVisionTasks(".../wasm");
const face = await FaceLandmarker.createFromOptions(vision, {
  baseOptions: {
    modelAssetPath: ".../face_landmarker.task",   // ~4 MB
    delegate: "GPU",                               // or "CPU"
  },
  numFaces: 1,
  outputFaceBlendshapes: true,                     // the 52 expression channels
  outputFacialTransformationMatrixes: true,        // head-pose matrix
  runningMode: "IMAGE",                            // or "VIDEO"
});

const result = face.detect(imageElement);
// result.faceLandmarks         → [ [ {x,y,z} × 478 ], … ]  (normalized 0–1)
// result.faceBlendshapes       → [ { categories:[ {categoryName,score} × 52 ] }, … ]
// result.facialTransformationMatrixes → [ { data:[16 floats] }, … ]

This page loads the model through the showcase's shared auto-init loader and draws the mesh on a canvas at the image's natural resolution; for the webcam it switches the task to VIDEO mode and calls detectForVideo each frame.
