Basics: box every object

Detection in its simplest form. Give YOLOS-Tiny an image, and its Vision Transformer draws a labelled box around every object it recognises, each with a confidence score. Slide the threshold to decide how sure it has to be before a box counts.

The model runs once; the slider just filters the boxes it already found. Raise it and only the confident boxes survive; lower it and more (noisier) boxes appear.
