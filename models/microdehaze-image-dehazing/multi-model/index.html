Dehaze, then detect what the fog hid

Haze doesn't just look bad — it blinds downstream vision. Here MicroDehazeNet clears the image first, then the on-device YOLOv10 detector runs on both the hazy and the cleared version, so you can see the objects the fog was smothering. Two models, chained, entirely local.

Run it

Two models load below (MicroDehazeNet ~4 MB, YOLOv10n ~9 MB). Pick an outdoor sample, haze it, then run the pipeline — we detect on the hazy image and on the cleared one so you can see the gain.

Both models run on-device; the image, the cleared result and the boxes never leave the tab.

Why chain them

Dehazing is a force-multiplier for downstream vision: a detector, OCR model or classifier all see cleaner structure once the atmosphere is removed. This is the on-device version of a classic "see through the fog" pipeline for dashcams, drones and surveillance — no server, no upload, each stage a real model running in your browser.
