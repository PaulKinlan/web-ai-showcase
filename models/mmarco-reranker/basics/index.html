Basics: which passage answers the query?

The core move: give the cross-encoder a query and a set of candidate passages, and it scores each one by how well it answers the query — then you sort by that score. Because it reads the query and passage together on the XLM-RoBERTa backbone, it judges genuine relevance, not surface similarity — across 14 languages.
