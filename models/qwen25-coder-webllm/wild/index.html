Wild: translate & refactor

A code model doesn't just write from scratch — it transforms. Paste a function and either translate it into another language (idiomatically, not line-for-line) or refactor it (cleaner, faster, or in a different style). Watch the rewrite stream on your GPU. Great for porting snippets, learning a new language by example, or modernising old code.
