Wild: a pocket text adventure

Something you couldn't easily do before small models ran in a tab: LFM2 plays game master for a text adventure, narrating the world and reacting to what you type, keeping the thread across turns — all locally. Pick a setting, then type actions like "open the door" or "look under the rug".

backend – dtype – turn time – turns 0

The whole game state lives in the conversation history: each of your actions and the model's narration is appended, so LFM2 "remembers" the story so far. No database, no server — the adventure exists only in this tab.
