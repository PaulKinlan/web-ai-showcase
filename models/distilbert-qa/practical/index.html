Practical: a FAQ bot over text you control

Paste any document you own — a returns policy, an onboarding doc, a set of meeting notes — and ask it questions. The model reads the whole thing and returns the exact sentence fragment that answers you, so support and self-serve answers stay grounded in your text. Nothing is uploaded.

This is single-passage QA: it works best when the answer lives in one place in the text. For long documents split across many sections, first retrieve the right paragraph (see the multi-model demo), then extract from it.
