GTCRN — strip the noise out of speech

A different audio task from the ASR, TTS, and classifier demos: speech enhancement. GTCRN (Grouped Temporal Convolutional Recurrent Network) is a tiny real-time denoiser — ~48k parameters, a 0.5 MB ONNX — that takes the noisy speech spectrum frame by frame and predicts the clean spectrum, which we invert back to audio. Add noise to a clean clip (so the gain is measurable) or bring your own noisy recording, and hear it cleaned up — entirely on your device.

Why it matters

A 0.5 MB denoiser that runs faster than real time on a CPU makes clean audio a local, private, offline primitive — no upload, no service, no GPU.

  • Calls & meetings — clean up a microphone before it hits a WebRTC call, in the browser.
  • Voice notes & dictation — denoise before recording or before speech-to-text, improving accuracy.
  • Accessibility — make speech clearer for hard-of-hearing listeners, on any device.
  • Field recording & archives — batch-clean noisy recordings without sending them anywhere.

Use cases

Basics

Add noise to a clean clip, enhance it, and hear + measure the before/after gain.

Practical

A voice-note cleaner: record or upload your own noisy audio and download the enhanced WAV.

Wild

Noise stress test: sweep the input SNR and watch how much GTCRN recovers at each level.

Multi-model

Denoise → Whisper: transcribe noisy vs enhanced audio and compare the two transcripts.

How the API works

There's no transformers.js speech-enhancement class, so we run the ONNX graph directly and do the STFT/iSTFT ourselves:

import * as ort from "onnxruntime-web";
const gtcrn = await ort.InferenceSession.create(bytes);         // ~0.5 MB

// per STFT frame (512-pt, hop 256, sqrt-Hann), carrying the recurrent caches:
let conv = zeros, tra = zeros, inter = zeros;
for (const frame of stftFrames(noisy16k)) {          // frame = [1,257,1,2] real/imag
  const o = await gtcrn.run({ mix: frame, conv_cache: conv, tra_cache: tra, inter_cache: inter });
  conv = o.conv_cache_out; tra = o.tra_cache_out; inter = o.inter_cache_out;
  pushEnhancedFrame(o.enh);                            // [1,257,1,2] → iSTFT overlap-add
}

This page runs exactly that in a Web Worker (STFT + streaming GTCRN + iSTFT all off the main thread), and transfers the enhanced PCM back for playback + download.
