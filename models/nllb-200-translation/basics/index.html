Basics: translate a phrase into any of 200 languages

The simplest thing NLLB does: take a sentence in one language and render it in another. Type a phrase, search the target across all 200 languages, and translate. Then change just the target and translate again — one model, any of 200 languages, including many other tools skip.

NLLB translates directly between the pair you pick — it doesn't secretly pass through English. Try the same sentence into three different targets and you're using one model, not three.
