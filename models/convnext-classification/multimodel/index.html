Multi-model: ConvNeXt + ViT ensemble

Two architectures make different mistakes — so averaging them cancels noise. Run the same image through the ConvNet and the transformer, average their softmax distributions, and the ensemble is usually more robust and better-calibrated than either alone. The agreement between them is a free confidence signal. Loading ViT downloads ~88 MB the first time.
