Wild: one model, many languages

The full StarCoder family was trained on 80+ programming languages — but the tiny checkpoint that runs in a browser (tiny_starcoder_py) was trained on Python. That makes this a live experiment: fill the same shape of gap across JavaScript, Rust, Go, SQL and more, and watch where a Python-specialised model still gets the idea and where it visibly strains. The difference is the lesson — it's exactly why a code model's language coverage matters. Every completion is real, streamed on your device.
