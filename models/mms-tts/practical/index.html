Practical: one recipe, many languages

MMS ships the same VITS architecture as a separate ~38 MB checkpoint per language — over a thousand of them, many with no commercial TTS at all. Pick a language below; its model downloads and caches on first use, then speaks. Switch back and forth freely — languages you've already loaded stay ready.

The example fills in the chosen language; edit it to anything the script supports.

Each language is a distinct checkpoint (Xenova/mms-tts-<iso>) loaded through the shared model-loader, so its cache/download state is tracked independently. This is the MMS pitch in miniature: speech technology that scales to the long tail of the world's languages.
