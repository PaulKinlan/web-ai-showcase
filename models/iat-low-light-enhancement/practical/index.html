Why on-device

Photos are personal. Running the fix locally means the picture never leaves the device — no upload to a photo-editing service, no per-image cost, and it works offline once the ~0.4 MB model is cached. The same enhancement can run as a batch step over a whole folder in a static web tool with no backend.
