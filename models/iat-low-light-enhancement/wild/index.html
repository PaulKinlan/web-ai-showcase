How dark can it go?

Restoration has a floor: once a region is crushed to pure black, there is no signal left to recover. Sweep the darkness toward near-black and watch the recovered brightness stop keeping up — the breaking point of a rescue model, made visible.

Run it

Step the darkness up run-by-run. Each run enhances the current dark image and logs the input vs recovered brightness so you can see the gap open up.

Where it breaks

At mild darkness IAT recovers almost the full scene. As you approach near-black, the recovered brightness lags further behind — the shadows go muddy and noisy because the model is amplifying quantisation, not real detail. That gap is the honest limit of any low-light model: it can lift buried signal, but it can’t manufacture light where none was recorded.
