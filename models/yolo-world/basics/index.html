Basics: type the classes, get the boxes

The whole idea of open-vocab detection in one screen. The class list is the vocabulary — edit it, detect, and swap a word to watch the boxes follow your language. There is no fixed 80-class list here; the words you type are the only classes the model looks for.

Try this

  • On the cats photo, start with cat, remote, detect, then add book and detect again — a new box class appears with no reload.
  • Replace cat with kitten or pet: the same animals still box, because CLIP understands the words are related.
  • Type something that isn't there (elephant) — you get no boxes, honestly. Open-vocab doesn't mean hallucinating.
