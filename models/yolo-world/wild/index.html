Wild: rare classes and prompt A/B

Two things a fixed detector can never do. First, detect objects that were never in any training list — type an oddly specific class and see if the model finds it. Second, run the same image against two phrasings of the same thing and watch how wording moves the boxes: "swan" vs "white bird on water", "sofa" vs "couch". Open-vocab detection is prompt-tunable, and this is where that gets fun.
