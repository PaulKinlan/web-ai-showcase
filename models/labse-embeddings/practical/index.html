Bitext mining — align sources to translations

The real business use: you have two piles of text — source sentences and a shuffled set of candidate translations in many languages — with no alignment metadata. LaBSE pairs each source with its true translation purely by cosine in its shared space. This is exactly how billion-scale parallel corpora are mined from the web to train machine-translation systems. Here the candidates carry a hidden gold order, so we can score accuracy against the truth.
