Basics: box every object

Detection in its simplest form. Give RT-DETRv2 an image and it draws a labelled box around every object it recognises, each with a confidence score — and because it's NMS-free, you get one clean box per object with no suppression tuning. Slide the threshold to decide how sure it has to be.

The model runs once; the slider just filters the boxes it already found. Raise it and only the confident boxes survive; lower it and more (noisier) boxes appear.
