Practical: triage an inbox into queues

The most shipped use of zero-shot: intent routing. Paste one message per line and a set of queues you define; the model sends each message to the queue it most entails. Edit the queue names and it re-routes — no retraining, no labelled data, nothing leaving the browser. This is the exact task a real support desk automates.

backend –messages –latency –

Each message is routed to its single best-entailed queue. A low top score (shown on the chip) is a useful signal for "needs a human" — you can threshold it to hold ambiguous messages back.
