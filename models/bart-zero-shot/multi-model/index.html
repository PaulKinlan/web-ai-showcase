Multi-model: route, then run the chosen model

Zero-shot makes a great router. Here BART-large-MNLI picks a coarse bucket for the text (stage 1). The winning bucket selects a different, fine-grained label set — and a second, much smaller model, DeBERTa-v3-xsmall, does that fine classification (stage 2). It's a real two-model cascade: the expensive model decides what kind of thing this is, and a cheap model handles the details of just that kind. Both run locally, nothing uploaded.

The first pass (BART) is the heavy one — it picks the domain. Each domain has its own sub-labels, and the second pass (DeBERTa, ~18× smaller) resolves them quickly. Loading DeBERTa adds a one-time ~70 MB download on your first "Route & classify".
