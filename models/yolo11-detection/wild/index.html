Wild: live webcam detection

YOLO11-nano is small and fast, and the NMS runs in microseconds on the CPU — a natural fit for a live camera. Start your webcam and it boxes what it sees, frame after frame, deduping each one with NMS. Every pixel stays on your device. Watch the per-frame latency and FPS as it runs.

backend – frame latency – fps – objects –

Detection runs in a Web Worker and the loop is driven by requestAnimationFrame, so the UI stays responsive and processing pauses automatically when the tab is in the background.
