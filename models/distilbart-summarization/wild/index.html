Wild: boil it down to one sentence

What happens if you summarize the summary — and keep going? Each pass feeds the previous result back in with a smaller budget, so the text collapses round after round toward its single most essential sentence. You get to watch information being distilled in real time.

rounds –start→end tokens – backend –total time –

This is a genuine fixed-point search: the loop stops when a pass stops shrinking the text, or when what's left is already a single short sentence. Some ideas resist compression — watch which details survive to the end. Each round is a real, separate model call.
