Multilingual cloze — mask a word, get an in-language completion

A genuinely useful (if rough) tool out of the raw checkpoint: write a sentence in any language, click a word to blank it out, and mT5 fills that blank in the same language. Because the vocabulary is shared across 101 languages, you do not pick a language anywhere — the model infers it from the surrounding words. This is how you would build an offline, private "suggest a word" feature with a single model.

Tip: mask a content word (a noun or verb) for the clearest results — masking "el/la/the" gives the model very little to reconstruct. It is a small, quantized, un-fine-tuned model, so treat the suggestion as a rough hint, not an oracle.
