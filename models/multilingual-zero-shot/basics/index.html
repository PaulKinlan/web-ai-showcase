Basics: label text in any language with categories you invent

The simplest thing multilingual zero-shot does: read text in one of many languages, take the labels you type, and score how well each fits — no training. Switch languages with the buttons; the model never changes. Edit the labels, or the hypothesis template, and watch the ranking move.

backend – latency – labels –

Single-label mode makes the labels compete (scores sum to 1). A hypothesis template written in the text's own language usually calibrates the scores better, because the NLI relation the model learned is aligned across languages.
