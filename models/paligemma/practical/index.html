Practical: detect objects by name

PaliGemma's detect prefix is open-vocabulary object detection with no bespoke detector: name any thing and the model emits <locNNNN> coordinate tokens. This page decodes those real tokens into boxes drawn over your image — labelled honestly as derived from the model's output.

Made by Paul Kinlan · Model runs locally in your browser by 🤗 Transformers.js · Source on GitHub
