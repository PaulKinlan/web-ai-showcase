Multi-model: a toxicity gate in front of analysis

Moderation is rarely the last step — it's the gate. Here two models run in sequence: toxic-bert screens each comment, and only the ones that come back clean pass through to a sentiment model (DistilBERT SST-2) for tone analysis. Toxic comments are held back and never reach the second stage — exactly how you'd protect a downstream pipeline from abusive input. Both models run locally.
