Corrupt it. It still encodes.

A vocabulary model has a brittle edge: feed it something it never tokenized and it fragments or breaks. A byte model has no such edge — every byte is valid. Take a sentence, then scramble characters, swap in look-alikes, flip random bytes, and stack emoji. The encoder never fails, and you can run ByT5 on the mess to see how gracefully (or not) it degrades. All local.
