Wild: cluster by meaning, ignore language

Here's something you couldn't do before this ran in a browser: hand BGE-M3 a pile of sentences in a jumble of languages and let k-means group them. Because the vectors live in one shared space, the clusters come out by topic — and each cluster turns out to contain the same idea written in several languages. The 2D map is coloured by cluster; the language flags just ride along.

The k-means and PCA run in plain JavaScript on the already-computed 1024-d vectors — the model's only job is to place same-meaning text near each other regardless of language, and it's the geometry that does the grouping. Nudge k and re-cluster to watch topics split and merge.
