Speech Commands × Whisper: wake, then transcribe

This is the pattern behind every voice assistant. A tiny, cheap keyword spotter listens for a wake word. Only when it actually hears it does it wake the expensive Whisper speech-to-text model to transcribe a full sentence — so you never run a heavy ASR model on every second of audio. Two local models, one gated pipeline, no server.

1 · Load both models

Speech Commands (wake-word spotter, ~83 MB)
Whisper (speech-to-text, ~120 MB)

2 · Choose a wake word and a trigger clip

The trigger clip is what the spotter hears. Say the wake word to wake Whisper; say anything else and it stays asleep.
