Wild: one line, every voice

SpeechT5's identity is a vector, so voice-swapping is trivial: hold the text fixed and run it through every speaker x-vector. You get the same sentence in each voice, side by side — with each speaker's fingerprint shown next to its clip. Something you could never do with a fixed-voice model.

backend –voices – total gen –

Every clip is a real, separate synthesis — same input text, different 512-dim x-vector. The accents and timbres are carried entirely by those vectors; the model and the words never change.
