Practical: read an article aloud

A private read-aloud feature. Paste text, pick a voice, and SpeechT5 speaks it one sentence at a time — the current sentence highlights as it plays. Because the model likes short utterances, splitting by sentence is exactly how you'd ship real read-aloud. Nothing is uploaded and there's no per-character bill.

Each sentence is a separate, real synthesis call — the highlight tracks the one currently playing. This is the pattern behind an accessible "listen to this page" button that never sends a word to a server.
