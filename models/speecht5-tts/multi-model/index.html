Translate, then speak it aloud

Two browser models in one pipeline. M2M100 translates text between languages; then SpeechT5 reads the translation aloud in a voice you choose. Great for turning a foreign passage into something you can hear in English — all on-device, nothing uploaded.

Each model initialises on its own below (cached copies auto-start, no button).

SpeechT5 is trained on English, so it reads English most naturally — translating into English gives the best-sounding result.

Why this is the interesting part

Translation and speech are each useful alone. Chained, they're a template for a whole class of tool: a travel phrasebook that speaks, an accessible reader for foreign news, a language-learning aid that reads translations back to you — all running with no server, no API bill, and no text leaving the tab. Swap Whisper in at the front and you've got spoken-language input too.
