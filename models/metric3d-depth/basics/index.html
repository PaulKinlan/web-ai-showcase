Basics · the fundamental capability

One photo → a metric depth map

The plain version: give Metric3D a photo and it returns a per-pixel depth map in metres, shown as colour with a legend in real distances. Cooler/darker is nearer, warmer/brighter is farther. Everything runs in your browser.

What you're seeing

  • The colour of each pixel is its distance in metres, mapped through the legend.
  • Because Metric3D is metric, the legend shows real distances — not arbitrary 0–1 units.
  • The whole per-pixel colour composite runs in a Web Worker, so a big photo never freezes the page.
