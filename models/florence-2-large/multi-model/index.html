Multi-model: read then reason

One model rarely ships alone. Here two run in a chain, both on your GPU: Florence-2 reads the words in an image (OCR) or describes the scene, and its text output becomes the input to Qwen2.5-0.5B, which summarizes it or answers your question. Vision → language → reasoning, entirely on-device.

This demo chains two models — each loads independently below, and both auto-init if already cached.
