Same photo, five separately trained networks. Run them all and see how Candy, Mosaic, Udnie, Pointilism and Rain Princess each rework the exact same scene — the clearest way to see that every "style" here is its own model, not a slider on one filter.

What it tells you

These are the original fixed-style networks: one model per painting. Seeing five of them diverge on one image makes the point tangible — the "look" lives in each network's weights, so the same content yields five genuinely different paintings, not five presets of one effect.
