Wild: domain vs general arena

Two BERTs enter, same masked sentence, same cased vocabulary — the only difference is what they read during training. Bio_ClinicalBERT studied MIMIC-III clinical notes; bert-base-cased studied Wikipedia and books. Type any sentence with a [MASK] and see where they agree, where they split, and how much more sharply the specialist commits on in-domain text. It's the domain gap, made measurable.

backend –both models –

Both models run in one Web Worker via AutoModelForMaskedLM, so they're exercised identically. Loading general bert-base-cased downloads ~132 MB the first time; Bio_ClinicalBERT auto-initialises above.
