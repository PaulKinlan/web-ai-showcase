Practical: on-device text tools

The same tiny model, three shippable jobs a developer would actually wire into a product: rewrite text in a chosen tone, classify it into labels you define, and extract structured fields. All local — the source text (often sensitive) never leaves the device, and there's no per-call bill.

The result streams here.

How it works

Each tool is just a different system prompt wrapped around the same model. Rewrite asks for a tone; classify constrains the answer to your label set; extract asks for one key: value line per field. That's the whole trick — an instruct LLM is a general-purpose text transformer you steer with instructions, no fine-tuning required.
