Wild: a tiny text-adventure agent

Let the model run a game. SmolLM2 plays dungeon master: it narrates a scene, then offers you a few choices; you pick one, and it continues the story — a self-contained agent loop that keeps its own state, entirely on your device. Set the scene, then play.

What makes this an "agent"

The page keeps a running transcript and re-sends it each turn, and the system prompt asks the model to always end with 2-3 concrete options. The buttons you see are parsed out of the model's own reply — so the loop is: model narrates & proposes actions → you choose → model continues. No server, no game engine; the LLM is the engine, deciding what happens next.
