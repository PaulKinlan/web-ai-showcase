Multi-model: rank the scene from nearest to furthest

Two models, one picture. SegFormer labels every pixel with a class (person, sofa, wall, plant…); Depth Anything V2 Large measures how far away each pixel is. Intersect them and you get something neither model produces alone: which object is in front of which — a distance-ordered list of the scene, computed entirely in your browser.

This page loads both models (~360 MB combined at q8, cached after first load). WebGPU accelerates depth where available; both fall back to WebAssembly and the page says so.

backend –depth –segment –objects –

For every class SegFormer found, we average the Large depth model's value over that class's pixels, then sort front-to-back. Sharper depth edges from the ViT-Large encoder make the per-object averages cleaner — a small object touching a distant background is less likely to borrow its neighbour's depth. Both models run once each in their own Web Worker; the ranking is canvas maths.
