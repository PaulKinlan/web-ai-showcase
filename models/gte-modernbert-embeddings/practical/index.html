Practical: long-document search & near-duplicate detection

Two workhorses. Semantic search: rank a set of passages by meaning against a query, not by keyword overlap. Dedup: find pairs that say the same thing in different words so you can collapse them. Both embed each passage once and reuse the vectors — and both handle long passages whole, thanks to the 8192-token context.
