Basics: a prompt in, a music clip out

The whole idea, plainly. Describe some music, choose how long, press Generate. MusicGen predicts a stream of audio tokens and decodes them to a 32 kHz clip on this device — no server in the loop. It's compute-heavy, so start short.

Compute-heavy. Runs on the CPU (WASM); even a few seconds takes real time. The progress bar tracks actual audio-token decoding.
