Basics: every label gets its own probability

Give SigLIP a picture and a list of words. Unlike CLIP, it doesn't force the words to compete — each one gets an independent 0–1 score. So an image can match several labels at once, or none. The two demos below make that concrete: first the scores themselves, then the exact same logits run through SigLIP's sigmoid versus CLIP's softmax, side by side.

2 · The same logits: sigmoid (SigLIP) vs softmax (CLIP)

This is the one idea that separates SigLIP from CLIP. Both compute a logit per label. SigLIP squashes each logit with an independent sigmoid; CLIP runs a softmax across the whole set. Run demo 1 above, then read across: the sigmoid column need not total 100%, the softmax column always does.
