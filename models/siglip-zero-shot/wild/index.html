Wild: an aesthetic meter, and a live vibe check

A softmax score is only meaningful relative to the other labels in the same call. SigLIP's sigmoid is an absolute 0–1 read on one label — so you can score many images on the same abstract quality and rank them, or point your camera at the room and ask whether it fits a vibe, live. Both run entirely on-device.

1 · Aesthetic / vibe meter

Pick a vibe word (or type your own). Each sample image gets an independent score for that one quality, so you can compare them on a shared scale — impossible with softmax.

2 · Live webcam vibe check

Grant camera access, type a target vibe, and SigLIP scores the live frame a few times a second. Nothing is uploaded — frames are grabbed to a canvas and scored locally. No camera? This section shows an honest unsupported state and the rest of the page still works.
