Wild: a cross-lingual concept meter, and an aesthetic meter

Two things you could never do with an English-only softmax classifier. First, a cross-lingual concept meter: write one concept in several languages and watch them all line up on the same image — proof that SigLIP 2 puts "cat", "chat", "gato" and "猫" in one shared space. Second, an aesthetic meter: SigLIP's sigmoid is an absolute 0–1 read, so you can score many images on the same abstract quality and rank them. Both on-device.

1 · Cross-lingual concept meter

Pick an image, then write the same concept in as many languages as you like. Each translation is scored independently — if they cluster near the top, the concept survived translation inside the model's shared space.

On the cats image, every translation of "cat" should sit well above every other concept — the model never saw a translation table, it just learned a shared multilingual space. (Homographs like the French "chat", which also means "online chat", can dip — a fun edge to probe.)

2 · Aesthetic / vibe meter

Pick a vibe word — in any language — and each sample image gets an independent score for that one quality, so you can compare them on a shared scale. Impossible with softmax.
